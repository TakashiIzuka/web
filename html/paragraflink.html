Hyperlink Dalam HTML

Hyperlink dalam HTML digunakan untuk membuat tautan yang memungkinkan pengguna untuk berpindah dari satu halaman ke halaman yang lainnya atau ke lokasi tertentu dalam halaman yang sama.

Link di bagi menjadi 8 setau saya di bawah ini.
Jenis-jenis Link:

  1. Link Absolut

    2. Link Absolut adalah tautan (link) yang merujuk ke halaman di situs web lain, dan alamatnya harus di tulis dengan lengkap, termasuk domain dan jalur yang benar.
    Contoh Link:

    Link Absolut
  2. Link Relatif

    3. Link Relatif adalah tautan yang menghubungkan satu dokumen dengan dokumen lain dalam situs web yang sama atau juga bisa menghubungkan satu file/halaman html dengan file html yang lain.
    Contoh Link:

    Kembali Ke Awal
  3. Link Internal / Anchor

    4. Link internal dikenal sebagai link dalam satu halaman adalah jenis tautan yang menghubungkan bagian-bagian berbeda dari halaman web yang sama.
    Contoh Link:

    Ke Bawah
  4. Target Link

    5. Target link mengacu pada cara browser menampilkan halaman yang dituju saat pengguna mengklik suatu link.
    Contoh Link:

    Link target
  5. Link Telepon

    6. Link telepon dapat digunakan untuk melakukan panggilan telepon. Jika link ini di klik, makan melakukan panggilan telepon pada nomor tujuan.
    Contoh Link:

    Link telepon
  6. Link Email

    7. Link Email digunakan untuk mengirim ke alamat email. Jika link ini diklik, maka akan membuka jendela pengirim email dari program email yang terinstall pada komputer.
    Contoh Link:

    Kirim Email
  7. Link SMS

    8. Link SMS digunakan untuk melakukan pengiriman pesan SMS pada nomor tujuan. Jika link ini diklik, maka akan melakukan pengiriman SMS pada nomor tujuan. SMS bisa juga di ubah menjadi pesan WhatsApp.
    Contoh Link:

    Kirim SMS
    Kirim WA
  8. Link Download

    9. Link download digunakan untuk mengunduh file/data dari server atau sering di sebut dengan istilah download. Jika link ini diklik, maka akan muncul kotak dialog yang menanyakan apakah anda akan menyimpan file itu dalam harddisk atau membukanya dengan program aplikasi yang sesuai.
    Contoh link:

    Download Script
