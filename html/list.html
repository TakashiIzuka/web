Cara Bermain GTA 5 di HP

  • Alat dan Bahan:
    • Alat:
    - HP Spek Tinggi.
    - Keyboard.
    - Mouse.
    - Tripod.
    - OTG.
    - USB Extetnd.
    - Gamepad (opsional).
    Bahan:
    - Winlator
    - File Game GTA 5
  • Langkah- langkah:
    • Penginstalan dan ekstrak file:
    1. Install aplikasi Winlator.
    2. Ekstrak file GTA 5 di folder download.
    3. Buka aplikasi Winlator dan klik tombol + di pojok kanan atas untuk menambah container.
      Kemudian Klik tombol play di container agar masuk ke tampilan dekstop.
    4. Setelah masuk ke dekstop, nanti akan muncul File explorer.
      Lalu cari Folder D nanti akan muncul folder GTA 5 jika kalian ekstrak nya di folder download.
    5. Lalu buka folder kemudian double klik di file exe GTA 5.
    6. Selamat anda sudah bisa bermain GTA 5 di hp.
